## Page Object Model

Encapsulate selectors and page behaviors so tests describe intent rather than DOM trivia.

:::info
Keep assertions in the test layer and keep page objects focused on interaction primitives.
:::

#### Example object

```ts
export class LoginPage {
  constructor(private readonly page: Page) {}

  email = this.page.getByLabel('Email');
  password = this.page.getByLabel('Password');
  submit = this.page.getByTestId('login-button');

  async signIn(user: { email: string; password: string }) {
    await this.email.fill(user.email);
    await this.password.fill(user.password);
    await this.submit.click();
  }
}
```
