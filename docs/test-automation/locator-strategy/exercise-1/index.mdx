# Test Automation

## Locator Strategy

Pick selectors that are stable across UI refactors and easy to maintain.

:::info
A data-testid selector is often easier to maintain than a fragile XPath.
:::

### Exercise 1. Locator Strategy

#### Scenario

Select the most stable locator for a login button.

#### Good pattern

```ts
button[data-testid="login-button"]
```
