# Document Site Overview

This site is a structured collection of software testing exercises, sample answers, and practice notes.

It is designed to help you review testing techniques in a consistent format, with each exercise presented as a single MDX page and supporting images stored alongside the exercise itself.

## What You Will Find

- Chapter-level overviews for major testing topics
- Individual exercises with questions and sample answers
- Tables, formulas, and diagrams rendered directly in Docusaurus
- Exercise-specific images kept inside each exercise folder

## Site Structure

- `Home` introduces the site and explains how to use it
- Each chapter has its own overview page
- Each exercise lives in a dedicated folder under its chapter
- Each exercise folder can contain:
  - one MDX file named `index.mdx`
  - an `images/` folder for illustrations and diagrams

## How To Use This Site

1. Open a chapter from the sidebar.
2. Read the chapter overview to understand the topic scope.
3. Select an exercise from the chapter list.
4. Review the question first, then open the sample answer if needed.
5. Use the images and tables inside the exercise page as reference material.

## Current Chapters

| Chapter | Purpose |
|---|---|
| [Black Box Testing](black-box-testing/) | Exercises on black-box testing techniques such as Boundary Value Analysis and related approaches |
| [Test Automation](test-automation/) | Exercises and notes related to automation workflows and implementation practice |

## Design Goals

- Keep the navigation compact and easy to scan
- Make it simple to add hundreds of exercises without reorganizing the whole site
- Keep each exercise self-contained so the question, answer, and images stay together
- Use a predictable structure that is easy to extend over time
