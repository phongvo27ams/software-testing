# Exercise 1. Equivalence Partitioning

## Scenario

Shipping is free for orders above 100 USD.

### Classes

- valid free-shipping orders
- valid paid-shipping orders
- invalid orders

### Formula

$$
fee = price - discount
$$

### Table

| Class | Example | Expected |
| --- | --- | --- |
| Free shipping | 120 | No fee |
| Paid shipping | 80 | Fee applies |
| Invalid | -1 | Reject |
