# Black Box Testing

Black-box testing exercises focus on validating behavior from the outside, based on requirements and expected outputs rather than implementation details.

## Topics

| Topic | Purpose |
|---|---|
| [Boundary Value Analysis](boundary-value-analysis/) | Exercises that derive test cases from numeric boundaries and nominal values |
| [Equivalence Class Partitioning](equivalence-class-partitioning/) | Exercises that divide input domains into valid and invalid classes |

Use the topic pages from the sidebar to open the exercises under each technique.
