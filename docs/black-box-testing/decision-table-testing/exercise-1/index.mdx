#### Decision matrix

| Member | Coupon | Weekend | Result |
| --- | --- | --- | --- |
| Yes | Yes | Yes | 25% discount |

| No | Yes | No | 5% discount |

#### Lightweight rule

$$
price_{final} = price_{base} - discount
$$

#### Sample code

```ts
type BookingRule = {
  isMember: boolean;
  hasCoupon: boolean;
  isWeekend: boolean;
};

function calculateDiscount(rule: BookingRule) {
  if (rule.isMember && rule.hasCoupon && rule.isWeekend) return 0.25;
  if (rule.isMember && rule.isWeekend) return 0.1;
  if (rule.hasCoupon) return 0.05;
  return 0;
}
```
